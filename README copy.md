# Zitate
Um einem Text zusätzliche glaubwürdigkeit zu verleihen, referenzieren wir auf:
> Ein Zitat

Ebenfalls kann man in einem Satz `Code hervorheben`um ihn speziell herauszuheben.

Mehrere Zeilen Code stellt man so dar:
```
Hier stehen
mehrere Zeilen
Code am Stück
```
